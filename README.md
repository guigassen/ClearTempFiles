# ClearTempFiles
A project to clear temp files using TopShelf.


To install service:

* Build project in Debug ou Release Mode
* Enter in bin/Debug or bin/Release folder
* Run cmd.exe or PowerShell as administrator
* Run .\ClearTempFiles.exe install
* Run .\ClearTempFiles.exe start

To uninstall service:

* Enter in bin/Debug or bin/Release folder
* Run cmd.exe or PowerShell as administrator
* Run .\ClearTempFiles.exe uninstall
